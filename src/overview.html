Cet ensemble de package et de classes forme une adaptation du jeu de cartes ShapeUp en version virtuelle. Le jeu est jouable à la fois dans la console et dans l'interface graphique. Différentes options sont disponibles pour le jeu : - 2 types de joueurs : virtuel et réel - 3 formes de tapis : Triangle Rectangle, Rectangle 3x5, Disque - 3 modes de jeu : Base, Avancé, Victoire Ennemie Les différents composants sont séparés dans 3 packages : Modele, Vue et Controleur, permettant de respecter le patron de conception MVC.
